##  Overview
Predicting RNA 3D structure is a challenging task that relies on identifying spatial patterns using only nucleotide sequences. In this notebook, a **baseline deep learning approach** is implemented to establish a reference performance for further experimentation and improvements.

##  Dataset
- Dataset: Stanford RNA 3D Folding (Kaggle)
- Input: RNA sequences
- Target: 3D structural coordinates
- Task Type: Regression / Structure prediction

##  Methods Used
- PyTorch-based deep learning pipeline
- Baseline neural network architecture
- Standard training and validation workflow
- Loss monitoring and evaluation metrics

##  Results

- Comparing advanced architectures

- Improving feature engineering and loss functions
